# Sistema de Control de Ventas

## Descripcion 

Este proyecto consiste en el desarrollo de un *Sistema de Control de Ventas* de una Tienda 

El sistema permite registrar, almacenar y analizar información de productos, incluyendo su nombre y precio, utilizando estructuras básicas de programación como **funciones, arreglos, ciclos y condicionales**.

Este proyecto fue desarrollado como parte del aprendizaje de los fundamentos de programación, aplicando conceptos de programación estructurada y lógica computacional.

## Objetivo

El objetivo principal es aplicar los conocimientos básicos de programación para desarrollar una solución funcional que permita gestionar información de productos y realizar operaciones matemáticas y de búsqueda sobre los datos almacenados.

## Funcionalidades principales

El sistema incluye las siguientes operaciones:

- Registrar productos
- Mostrar lista de productos
- Calcular el total de precios
- Buscar un producto por nombre
- Identificar el producto más caro
- Mostrar estadísticas generales de los productos
- Salir del sistema

## Tecnologías utilizadas
Lenguaje de programación: **C++**
Entorno de desarrollo: **Consola / Terminal**
